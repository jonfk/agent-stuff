---
name: propose-commit
description: Propose a git commit for the changes you worked on.
---

Given what you are currently working on and completed, propose a git commit message and body. Make sure to capture the intent of the change as well as what was done. 

Use conventional commits convention for the commit subject. Use last several commits to inform your proposal. The commit message length should be proportional to the importance of the change. A simple file change doesn't need a body. A critical change should have an explanation for why.

If there are multiple options that are actually different, propose me a choice.

Make sure to have my agreement before creating the commit.
