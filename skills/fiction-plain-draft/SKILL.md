# Fiction Plain Draft

Use this skill to clarify what a scene is doing, then draft the plainest useful version of it. The goal is readable structure, not final prose.

## Core rules

- Create a scene contract before drafting prose.
- Draft plainly.
- Prioritize sequence, causality, desire, conflict, turn, and consequence.
- Avoid lyricism, ornate metaphor, elaborate description, and final line polish.
- Good lines are allowed, but do not optimize for beauty yet.
- Do not add major facts that conflict with the codex.
- If the scene contract is unclear, make the smallest reasonable assumption and state it.
- Keep the draft no longer than needed to test the scene.

## Process

1. Read the codex or brief for relevant facts.
2. Write a compact scene contract.
3. Draft the scene in simple, direct prose.
4. List any durable codex updates created by the draft.

## Plain draft template

```md
# Scene Contract

Purpose:
POV:
Starting state:
Ending state:
Central tension:
Turn:
Required beats:
Do not solve yet:
Assumptions:

# Plain Draft

[Draft the scene here.]

# Review Notes

What changed:
What works:
What feels weak:
Possible cuts:
Open decisions:
```

## Plain draft style

Prefer:

- clear action
- simple scene geography
- direct dialogue
- visible choices
- emotional change shown through behavior

Avoid:

- decorative description
- vague interiority
- repeated emotional explanation
- invented backstory not required by the scene
- polishing sentences before the scene works
