# Fiction Revision

Use this skill after a plain draft exists. Its job is to improve the draft without losing sight of what the scene is doing.

## File behavior

Operate on one revised-draft file, usually:

- `revised-draft.md`
- `[story-name].revised-draft.md`

When creating a revised draft, use the file template below.  
The revised prose is the main output. Keep notes brief and place them after the draft.  
Do not include long diagnosis unless the user asks for it.

Use the template as a starting shape, not as mandatory bureaucracy. Omit empty sections when they do not help.

## Core rules

- Diagnose before rewriting unless the user asks for a direct revision.
- Use the lowest unfinished level of the revision ladder first.
- Prefer targeted changes over full rewrites.
- Do not polish sentences before the scene’s dramatic function is stable.
- Preserve the user’s strongest material where possible.
- Do not add major new plot facts unless requested.
- Keep revision notes short and practical.

## Revision ladder

Use the lowest unfinished level first:

1. **Drama** — desire, conflict, escalation, turn, consequence.
2. **Body** — gesture, action, setting, objects, sensory detail, silence.
3. **Voice** — narrator, character perspective, diction, tone.
4. **Music** — rhythm, sentence shape, emphasis, image pattern.
5. **Compression** — cuts, merges, sharper openings and endings.

## Process

1. Identify the draft’s primary issue.
2. Choose the next useful revision pass.
3. Revise only at that level unless asked otherwise.
4. Preserve what already works.
5. Add a compact note explaining the main change.

## Revised draft file template

```md
# Revised Draft

## Scene: [Scene name or number]

[Revised scene prose.]

### Revision Notes

Pass:
Changed:
Still needs attention:
```

## Pass guidance

### Drama pass

Strengthen the scene’s want, obstacle, pressure, turn, and consequence. Reorder or cut beats if needed.

### Body pass

Replace abstract emotion with concrete behavior, objects, setting, sensory detail, silence, and subtext.

### Voice pass

Make the prose sound more specific to the narrator, character, genre, and story world. Avoid generic literary smoothness.

### Music pass

Improve rhythm, sentence shape, emphasis, paragraph movement, and memorable phrasing.

### Compression pass

Cut repetition, throat-clearing, over-explanation, decorative description, and any beat that does not advance or complicate the scene.
